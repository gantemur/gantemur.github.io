# Launch checklist

Use this before treating the site as the public homepage.

## Identity and contact

- Verify current title and affiliation.
- Verify contact email.
- Verify location and office details, or remove office details if no longer useful.
- Confirm preferred public name order and Mongolian name form.
- Confirm GitHub username and repository spelling.

## Publications and CV

- Verify 2026 JHEP metadata for "Rotationally invariant dynamical lattice regulators for Euclidean quantum field theories".
- Verify arXiv and author metadata for "Exact Loop Controllers for ReLU Realization of Homogeneous Curve Refinements".
- Add DOI, issue, page/article numbers, and public links only after verification.
- Add current CV PDF/source, or decide to wait for generated CV output.
- Decide which publications are selected or featured on the homepage.

## Deployment

- Decide final repository name and remote, likely `gantemur.github.io`.
- Decide GitHub Pages URL.
- Confirm Pages source uses GitHub Actions.
- Run `npm run check` and `npm run build` before pushing.

## Old site and external projects

- Decide whether the old McGill site should redirect, remain as-is, or be linked as an archive.
- Decide what course pages, if any, will be migrated rather than only listed.
- Link Sol once the separate project exists.
- Link blog projects only after their URLs and scope are stable.
- Keep interactive tools, calendars, animations, and demos outside this homepage.
